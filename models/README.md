# 訓練模型
1. 用 Conda 或 Venv 啟用虛擬環境
2. 執行 `train.py`、`train.bat` 或 `train.sh` 
    ```shell
    python train.py
    ```
    ```shell
    train.bat # Windows 批次檔
    ```
    ```shell
    . ./train.sh # Shell Script
    ```
